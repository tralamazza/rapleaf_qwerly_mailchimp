# Mailchimp + Rapleaf + Qwerly

  This app will connect to your Mailchimp account, fetch every email in ALL your lists and update each one of them using RapLeaf and Qwerly (dynamically creating Mailchimp fields).

## Prerequisites

* MongoDB
* node.js
* npm

## Install and Run

Go to the app folder, open `app/config.js` and enter your API keys

    app/config.js

Install all dependencies

    npm install -d

And run it

    node index.js
